# ExoPaper UI 🌌

Фронтенд-приложение для платформы **ExoPaper RAG**, созданное на React 19, TypeScript и Vite. Интерфейс выполнен в премиальном футуристичном стиле "Deep Space" с использованием интерактивного 3D-движка на базе Three.js.

---

## 🎨 Дизайн и Особенности

- **Эстетика Deep Space:** Темные градиенты, полупрозрачные стеклянные панели (Glassmorphism), неоновое свечение и мягкая анимация.
- **Интерактивный 3D-холст:** Реалистичное трехмерное процедурное рендеринг-окружение звездных систем с плавающими орбитами, кольцами пыли, эффектами свечения звезд и радиальных возмущений (Stellar Jitter) для планет, открытых методом скоростей.
- **Real-time реактивность:** Автоматическая синхронизация через SignalR. Новые векторизованные статьи и теги появляются на лету в виде уведомлений и обновляют графики статистики без перезагрузки страниц.
- **Чистый код в React 19:** Соблюдение строгих правил React 19 Strict Mode (без случайных значений в циклах рендеринга Three.js, строгая очистка WebSocket-соединений).

## 📂 Структура папок

- `src/api/` — API-клиенты для работы с бэкендом (Axios).
  - `client.ts` — базовый клиент с настройкой baseURL и обработкой ошибок.
  - `exoplanets.ts`, `papers.ts`, `sync.ts` — типизированные методы для соответствующих ресурсов.
- `src/components/` — повторно используемые UI-компоненты:
  - `layout/` — разметка страниц, Sidebar, Header с индикацией SignalR.
  - `dashboard/` — интерактивные графики (DiscoveryChart) и панели событий.
  - `search/` — гибридная строка поиска с фильтрами.
  - `three/` — 3D-компоненты React Three Fiber (`PlanetMesh`, `StarMesh`, `OrbitRing`, `Starfield`).
  - `planet/` — детальные карточки параметров, графики неопределенности и связанные статьи.
- `src/hooks/` — кастомные React-хуки:
  - `useSignalR.ts` — управление жизненным циклом WebSocket-соединения с бэкендом.
- `src/pages/` — страницы-экраны:
  - `DashboardPage.tsx` — сводная статистика и глобальный поиск.
  - `PlanetDetailPage.tsx` — страница планеты со сценой 3D и аналитическим RAG-модулем.
  - `PapersPage.tsx` — каталог публикаций arXiv с поиском.
- `src/stores/` — Zustand-хранилища (`appStore.ts` для хранения логов событий SignalR).
- `src/types/` — TypeScript-типы для сущностей домена.

## 🚀 Запуск и Разработка

### Шаг 1: Установка зависимостей
Убедитесь, что у вас установлен **Node.js v20+**.
В папке `ExoPaper.UI` выполните:
```bash
npm install
```

### Шаг 2: Запуск в режиме разработки
```bash
npm run dev
```
По умолчанию приложение запустится на [http://localhost:5173](http://localhost:5173).

При локальной разработке все запросы к API проксируются через настройки Vite (см. `vite.config.ts`), либо обращаются по настроенному CORS адресу бэкенда.

### Шаг 3: Сборка для продакшена
```bash
npm run build
```
Статические файлы будут скомпилированы в папку `dist/`, готовую для раздачи через Nginx (конфиг `nginx.conf` настроен для работы в Docker-контейнере с проксированием `/api/` и `/hubs/`).
